## 3. CONVERSION-DRIVEN IMAGE STRATEGY

### Image Placement Specifications

#### 3.1 Hero Section - Main Lifestyle Image
**Placement:** Background or right side of hero card
**Dimensions:** 1200x800px (3:2 ratio)
**Style:** Warm, natural lighting, authentic moment

**Content Description:**
- **Primary Subject:** Parent (30s-40s) sitting with child (8-11 years old) at kitchen table
- **Emotion:** Child showing mild frustration (furrowed brow, slumped shoulders), parent leaning in with calm, attentive posture
- **Props:** Lemonade stand sign or craft supplies visible in background, worksheet/notebook on table
- **Lighting:** Golden hour natural light, warm tones
- **Framing:** Medium shot, slight overhead angle showing connection

**Conversion Psychology:**
- Shows the MOMENT (child upset, parent ready to help)
- Authentic emotion (not fake smiles)
- Relatability (kitchen table = real life, not studio)
- Visual proof of "10-minute routine" concept

**Alternative Stock Photo Sources:**
- Search terms: "parent child conversation frustrated," "mother daughter homework table," "father son problem solving"
- Avoid: Overly posed, fake enthusiasm, professional office settings

#### 3.3 The 5R Flow Section - Process Visualization
**Placement:** Between step cards or as header image
**Dimensions:** 1600x400px (4:1 panoramic)
**Style:** Illustrated infographic or photo sequence

**Content Description:**
- **5-panel visual sequence:**
  1. Child's face showing distress (tears, frustration)
  2. Parent placing hand on child's shoulder (connection)
  3. Child taking deep breath (eyes closed, chest rising)

**Conversion Psychology:**
- **Visual before/after** (panel 1 vs panel 5) shows transformation
- **Concrete process** makes abstract concept tangible
- **Non-threatening emotions** (realistic, not dramatic)

**Creation Options:**

  // Track scroll depth
  let scrollDepths = [25, 50, 75, 100];
  let scrollTracked = [];

  window.addEventListener('scroll', () => {
    let scrollPercent = (window.scrollY / (document.body.scrollHeight - window.innerHeight)) * 100;
    scrollDepths.forEach(depth => {
      if (scrollPercent >= depth && !scrollTracked.includes(depth)) {
        gtag('event', 'scroll', { depth: depth });
        scrollTracked.push(depth);
      }
    });
